# Sistem Manajemen Taekwondo

Project akademik berupa perancangan sistem manajemen taekwondo berbasis aplikasi desktop.

## Deskripsi
Sistem ini dirancang untuk membantu pengelolaan data pada klub taekwondo, meliputi data anggota, pelatih, peralatan, serta administrasi pembayaran dan laporan. Sistem memiliki dua jenis hak akses, yaitu Admin dan Anggota.

Project ini merupakan hasil perancangan sistem (analisis kebutuhan, alur proses, dan logika sistem), belum sampai tahap implementasi aplikasi secara penuh.

## Fitur Utama
### Admin
- Mengelola data anggota
- Mengelola data pelatih
- Mengelola data peralatan
- Mengelola laporan anggota
- Melihat dan mencetak laporan secara keseluruhan

### Anggota
- Melakukan pembayaran (SPP, kenaikan tingkat, dan kejuaraan)
- Melihat riwayat pembayaran
- Mencetak struk pembayaran
